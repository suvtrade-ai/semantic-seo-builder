# Semantic SEO Plugin Marketplace

Claude Code plugin marketplace by [suvtrade](https://github.com/suvtrade).

## Install

```
/plugin marketplace add suvtrade/semantic-seo-plugins
/plugin install semantic-seo-builder@suvtrade-seo
```

## Plugins

### semantic-seo-builder v0.8.0

Koray Tugberk Gubur Semantic SEO framework for local businesses.

**Track 1 — New site build (5 phases):**

- Phase 1: Foundation — customer voice research (blog-discourse), brand persona, competitor baseline (seo-plan)
- Phase 2: Semantic Research — entity/attribute matrices, SERP cluster validation (seo-cluster)
- Phase 3: Topical Map — hub-and-spoke architecture with intent validation (seo-sxo)
- Phase 4: Content System — brief → write → factcheck → schema → GEO → images
- Phase 5: Technical & Local SEO — full audit, schema (with floorLevel), local signals

**Track 2 — Ongoing operations:**

- Weekly pipeline: blog-write → factcheck → seo-check → schema → GEO → images → publish → GBP post → drift check
- Monthly audit: rankings, content refresh (blog-rewrite), AI citation tracking (blog-geo), backlink health (seo-backlinks)

**Integrations:** Works alongside claude-seo and claude-blog plugins for best results.
